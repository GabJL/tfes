## Descripción

La complejidad del código es uno de los principales factores que afectan a la mantenibilidad, la aparición de errores y el coste de evolución del software. Aunque existen herramientas capaces de medir diferentes métricas de complejidad, la aplicación de refactorizaciones sigue requiriendo habitualmente una importante intervención manual.

Este trabajo propone desarrollar un sistema híbrido que combine modelos de aprendizaje automático y modelos de lenguaje para automatizar el proceso de reducción de la complejidad del software. El sistema utilizará un modelo de aprendizaje automático para recomendar acciones de refactorización y un modelo generativo para aplicarlas automáticamente sobre el código fuente, validando posteriormente que las modificaciones preservan el comportamiento del programa mediante pruebas automatizadas y análisis estático.

Además, se realizará una comparación experimental frente a otras estrategias existentes de reducción automática de complejidad, analizando tanto la calidad de las refactorizaciones obtenidas como la eficiencia del proceso.

## Líneas principales

- Estudio de métricas de complejidad y técnicas de refactorización automática.
- Diseño de un modelo de aprendizaje automático para recomendar acciones de refactorización.
- Integración de modelos de lenguaje para aplicar automáticamente dichas acciones.
- Desarrollo de un proceso iterativo de validación mediante pruebas automatizadas y análisis estático.
- Comparación experimental con otros sistemas de reducción de complejidad.
- Evaluación de la calidad del código generado y de la reducción conseguida.
