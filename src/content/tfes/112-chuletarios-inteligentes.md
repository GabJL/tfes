## Descripción

Las competiciones de programación permiten utilizar un pequeño manual de consulta durante el concurso, conocido habitualmente como *chuletario*. Prepararlo requiere recopilar algoritmos, estructuras de datos y fragmentos de código útiles, organizarlos adecuadamente y adaptarlos a las restricciones de formato de cada competición. Este proceso suele ser manual, consume mucho tiempo y dificulta compartir conocimiento entre distintos equipos.

En este trabajo se propone desarrollar una plataforma web colaborativa donde los usuarios puedan aportar, revisar y clasificar implementaciones de algoritmos y otras referencias de interés. Además de servir como repositorio compartido, la plataforma permitirá generar automáticamente chuletarios personalizados adaptados a las reglas de distintas competiciones, facilitando su preparación y mantenimiento.

La inteligencia artificial podrá utilizarse como apoyo en diferentes tareas, como la traducción de código entre lenguajes de programación, la detección de posibles errores, la revisión de contribuciones o la generación de recomendaciones. El objetivo es construir una herramienta útil para la comunidad de programación competitiva, combinando desarrollo web, colaboración entre usuarios e integración de modelos de lenguaje.

## Posibles líneas de trabajo

- Diseñar e implementar una plataforma web colaborativa para la gestión de contenido.
- Desarrollar mecanismos de búsqueda, clasificación y etiquetado de algoritmos.
- Incorporar un sistema de votación y revisión de contribuciones.
- Integrar modelos de lenguaje para traducir código entre distintos lenguajes de programación.
- Diseñar herramientas de apoyo para la detección de contenido duplicado o potencialmente incorrecto.
- Implementar un generador de chuletarios adaptado a las restricciones de diferentes competiciones.
- Analizar el uso de la plataforma mediante estadísticas y recomendaciones personalizadas.

## Qué aprenderás

- Diseño y desarrollo de aplicaciones web.
- Diseño de plataformas colaborativas.
- Integración práctica de modelos de lenguaje (LLMs).
- Gestión y moderación de contenido generado por usuarios.
- Generación automática de documentos personalizados.
- Diseño y evaluación de aplicaciones software de cierta complejidad.
